CSV Format Guide
Required Columns:
  • Code: 4-10 digit account code (must be unique)
  • Name: Account name
  • Type: One of: ASSET, LIABILITY, EQUITY, INCOME, EXPENSE, HEADER
  • Currency: Currency code (e.g., GHS, USD, EUR)
Optional Columns:
  • Parent Code: Code of parent account (for hierarchical structure)
  • Description: Additional details about the account
Example CSV Format:
Code,Name,Type,Parent Code,Currency,Description
1000,Assets,HEADER,,GHS,Root asset category
1100,Cash,ASSET,1000,GHS,Cash accounts
1110,Cash at Office,ASSET,1100,GHS,Main office petty cash
2000,Liabilities,HEADER,,GHS,Root liability
2100,Accounts Payable,LIABILITY,2000,GHS,Vendor payables
Important Tips
  • Create parent accounts (HEADER type) before their child accounts
  • Ensure parent account codes exist before referencing them
  • Account codes must be unique across all accounts
  • Currency codes must exist and be active in the system
  • All accounts will be set as active by default
  • File must be UTF-8 encoded
  • Maximum file size: 5MB
