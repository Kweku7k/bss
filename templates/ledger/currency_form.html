Currency Guidelines
Currency Code:
  • Must be exactly 3 letters
  • Follows ISO 4217 standard
  • Examples: GHS, USD, EUR, GBP
Currency Symbol:
  • Used for display purposes
  • Can be Unicode symbols
  • Examples: ₵, $, €, £
Decimal Places:
  • Most currencies use 2 decimal places
  • Some use 0 (like Japanese Yen)
  • Some use 3 (like Kuwaiti Dinar)
Base Currency:
  • Only one currency can be base
  • Used for reporting and conversions
  • Usually the local currency
