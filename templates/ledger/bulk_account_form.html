CSV Format Guide
Required Columns:
  • Code: 4-10 digit numeric code
  • Name: Account name
  • Type: ASSET, LIABILITY, EQUITY, INCOME, EXPENSE, or HEADER
  • Currency: Currency code (e.g., GHS, USD, EUR)
Optional Columns:
  • Parent Code: Code of parent account
  • Description: Account description
Example:
Code,Name,Type,Parent Code,Currency,Description
1000,Assets,HEADER,,GHS,Root asset category
1100,Cash,ASSET,1000,GHS,Cash accounts
1110,Cash at Office,ASSET,1100,GHS,Main office petty cash
2000,Liabilities,HEADER,,GHS,Root liability category
2100,Accounts Payable,LIABILITY,2000,GHS,Vendor payables
Tips
  • Create header accounts first, then child accounts
  • Use consistent numbering (1000, 1100, 1110, etc.)
  • Parent codes must exist before creating child accounts
  • Account codes must be unique
  • All accounts will be created as active by default
